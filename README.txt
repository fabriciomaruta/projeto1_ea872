##READ-ME
O jogo conta com um makefile que abre o jogo automaticamente.

Como jogar?
Para jogar, execute o makefile e aguarde o jogo abrir na tela.

Como Controlar?
Para controlar o personagem (@) use as Teclas W,A,S,D para se movimentar, e a barra de espaço para atirar.
W - Pra frente
S - para baixo
D - Direita
A - Esquerda

Como ganhar?
A meta do jogo é ganhar do inimigo que se movimenta na tela (*). Existem ainda no mapa pontos para serem pegos (.), ao pegá-los seu personagem atira mais rápido. 
Para derrotar o inimigo, é preciso enfraquece-lo, acertando um tiro nele. Voce saberá que ele está vulnerável quando ele se transformar (+). Após ele estar vulnerável, é só passar por cima dele.
CUIDADO: Voce perde o jogo se ele te pegar primeiro e não estiver vulnerável.
